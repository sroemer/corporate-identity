# corporate-identity

## Corporate identity of SROEMER Software Engineering

This repository contains basic information about colors and fonts
as well as logos and templates used by SROEMER Software Engineering

### Logo

The company logo consists of the main company name 'sroemer' with the text
'software engineering below it. Both parts are separated by a simple arc.

The font used in the logo its
[Anita Semi Square](https://fontlibrary.org/en/font/anita-semi-square)
by Gustavo Paz. Following colors are used for the logo and its background:

+ blue (background): #003060
+ light (text)     : #0080C0
+ yellow           : #FFDE80
+ grey             : #606060
